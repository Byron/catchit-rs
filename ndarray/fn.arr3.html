Function ndarray::arr3 [−] [src]

pub fn arr3<A: Clone, V: Initializer<Elem=U>, U: Initializer<Elem=A>>(xs: &[V]) -> Array<A, (Ix, Ix, Ix)>

Return a three-dimensional array with elements from xs.

Panics if the slices are not all of the same length.

use ndarray::arr3;

let a = arr3(&[[[1, 2],
                [3, 4]],
               [[5, 6],
                [7, 8]],
               [[9, 0],
                [1, 2]]]);
assert!(
    a.shape() == [3, 2, 2]
);
